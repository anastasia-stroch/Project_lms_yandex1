Мой проект будет скорее игрой, чем просто приложением. В нём каждый раз будут рандомно в зависимости от сложности (от 1 до 3 уровня) составляться рабочие цепочки из логических элементов: AND, OR, NAND, NOR, XOR - цепочки будут видны пользователю и ему нужно будет вводить в поле для ввода какой результат будет на каждом из 3х заданных выходов в конце цепочки, далее система будет проверять ответ и приступать к следующей цепочке, если эта пройдена. Увеличивается уровень сложности тогда, когда пользователь три раза подряд правильно решает цепочку.
